# mouse

This library allows you to programmatically control the mouse. 
```
nimble install mouse
```

# Suported platforms
| Platform  | Support  | Implementation |
| ------------ | ------------ | ------------ |
|  Windows  |  full  |  WINAPI  |
|  Linux  |  full  |  X11  |
|  Mac  |  none |  none  |
